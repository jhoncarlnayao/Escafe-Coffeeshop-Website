## FAQ



### What is Escafe?

Escafe is a concept coffee shop website focused on storytelling, atmosphere, and modern café branding through immersive slide-based design.

### What technologies were used?

The project is built using:

* HTML5
* Tailwind CSS
* JavaScript

### Is this a real coffee shop website?

This project can be used as a concept design, portfolio piece, startup presentation, or foundation for a real café website.

### What makes this project unique?

Escafe focuses on cinematic layouts, elegant typography, smooth transitions, and emotional storytelling instead of traditional café website structures.

### Can I customize the design?

Yes. You can easily modify:

* Colors
* Images
* Typography
* Animations
* Slide content
* Layout sections

### Is the website responsive?

Yes, the layout is designed to adapt across different screen sizes.

### Can I use this project for my own café or business?

Yes, you can use it as inspiration or customize it for personal and commercial projects.

### Does this project use a framework?

The project mainly uses Tailwind CSS for styling and utility-based UI development.

### How do I run the project locally?

Simply open the project folder and run the `index.html` file in your browser, or use a local development server like Live Server in VS Code.

### Is this open source?

You can make it open source by uploading the project to GitHub with your preferred license.
